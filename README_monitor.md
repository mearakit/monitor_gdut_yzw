# 广东工业大学研究生招生网监控脚本

## 功能说明

此脚本用于监控广东工业大学研究生招生信息网（硕士招生页面）的更新情况：
- **监控网址**: https://yzw.gdut.edu.cn/sszs.htm
- **监控频率**: 默认每小时检查一次（可配置）
- **通知方式**: 通过QQ邮箱发送邮件通知

## 环境要求

- Python 3.6+
- 依赖包: `requests`, `beautifulsoup4`

## 安装依赖

```bash
pip install requests beautifulsoup4
```

## 配置说明

在脚本文件 `monitor_gdut_yzw.py` 中，找到**配置区域**，修改以下参数：

### 1. 邮箱配置（必填）

```python
SENDER_EMAIL = "your_qq@qq.com"      # 发件人QQ邮箱
SENDER_PASSWORD = "your_auth_code"    # QQ邮箱授权码（不是登录密码！）
RECEIVER_EMAIL = "receiver@qq.com"    # 收件人邮箱
```

### 2. 获取QQ邮箱授权码

1. 登录 [QQ邮箱网页版](https://mail.qq.com)
2. 点击顶部 **设置** → **账户**
3. 找到 **POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务**
4. 开启 **SMTP服务**
5. 按照提示获取 **授权码**（16位字符串）

### 3. 可选配置

```python
CHECK_INTERVAL = 3600  # 检查间隔（秒），默认1小时
DATA_FILE = "monitor_data.json"  # 存储历史数据的文件
```

## 使用方法

### 方式一：直接运行

```bash
python monitor_gdut_yzw.py
```

脚本会持续运行，按设定的时间间隔检查页面更新。

### 方式二：后台运行（Windows）

```bash
pythonw monitor_gdut_yzw.py
```

或使用计划任务：
1. 打开 **任务计划程序**
2. 创建基本任务，设置触发器为每小时
3. 操作选择 **启动程序**，程序路径填写 `python.exe`
4. 参数填写脚本完整路径

### 方式三：使用系统服务（Linux/Mac）

可以使用 `systemd` 或 `supervisor` 等工具将脚本作为后台服务运行。

## 注意事项

1. **首次运行**不会发送邮件，只会初始化数据记录当前页面状态
2. 请确保网络连接正常，脚本需要访问外网
3. 如果长时间运行，建议定期检查日志文件大小
4. QQ邮箱授权码如果泄露，请及时在邮箱设置中重置

## 常见问题

### Q: 脚本提示"请先在配置区域填写正确的QQ邮箱和授权码"
A: 请按照上述配置说明，在脚本中填写正确的邮箱和授权码。

### Q: 邮件发送失败
A: 请检查：
- QQ邮箱是否开启了SMTP服务
- 授权码是否正确（不是登录密码）
- 网络连接是否正常

### Q: 如何修改检查频率？
A: 修改 `CHECK_INTERVAL` 的值，单位是秒。例如改为每30分钟检查一次：
```python
CHECK_INTERVAL = 1800  # 30分钟
```
